Idiom # 41: Reverse a string

See programming-idioms.org:

Code

There is no feature in Fuzion's strings to reverse the characters, so we have to do it manually: First, the string that is usually represented as utf8 is converted to an array of codepoints. This array is then reversed and folded back into a string using string's concat monoid.
